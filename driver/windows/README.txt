VB-Cable Virtual Audio Device Installer
========================================

Bu klasore VB-Cable kurulum dosyasini indirip koymaniz gerekiyor.

ADIMLAR:
1. https://vb-audio.com/Cable/ adresine girin
2. "VB-CABLE Driver Pack" zip dosyasini indirin
3. Zip'i acin, icindeki "VBCABLE_Setup_x64.exe" dosyasini bu klasore kopyalayin
4. Klasor yapisi su sekilde olmali:
   driver/windows/VBCABLE_Setup_x64.exe

Uygulama, Windows'ta "Install Driver" butonuna tikladiginda
bu installer'i UAC yukseltmesiyle calistirarak VB-Cable'i kurar.
